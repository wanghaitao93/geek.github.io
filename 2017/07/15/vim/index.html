vim simple use

Posted on 2017-07-15 | In linux | |

vim

以下是我使用vim的总结，从配置到使用。

一、vim 的配置信息

1、以下vimrc配置可以基本满足使用

vi ~/.vimrc

添加一下信息:
  1 syntax on   //代码高亮
  2 set number  //行号
  3 set ts=4    //tab is space 4
  4 set expandtab   
  5 set autoindent
保存即可

2、该配置 .vimrc 适合程序开发使用，可以下载该文件到到本地的主目录下，把名字从vimrc改为.vimrc即可。

二、常用快捷键

按键 功能
h、j、k、l left、down、up、right
10j down 10 row
:123 移动到 第123行
gg 移动到 文本开头；
G 移动到 文本结尾
ctrl+f 向下翻页
ctrl+b 向上翻页

三、常用的功能

1、vim查找

/hello 
点击回车，按n向后查找关键词hello

?hello
点击回车，按n向前查找关键词hello

2、vim的全局替换

:%s/a/b
a替换成b

3、vim代码批量移动

v选中后，>>向后移动，<<向前移动
10>> 光标下面10行向后移动

4、从外界向vim复制代码,保证格式不变

:set paste

5、vim 打开新文件

vim 打开新文件到当前窗口
:open file
切换文件
Ctrl + 6

vim 打开新文件到新的窗口
:split(vsplit) file 
切换窗口 
Ctrl + ww

6、显示不可见的字符(\t、\n等)

:set list

\t    显示为    ^I
\n     显示为  $

插入\t    
Ctrl+ V  , I  然后Tab建
