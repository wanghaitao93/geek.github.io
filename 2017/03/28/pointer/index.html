C++ 指针的指针

Posted on 2017-03-28 | In C++ | |

基础知识

* ：解引用符（derefrence operator）
& : 取地址符 （address of）


int a = 1;
int *b = &a;
int **c = &b;

int *b        //声明b是一个指向int类型的指针.
&a            //a的地址
b = &a        //把a的地址赋给b

* 指针 = 对象的值
*b = a    = 1    

int **c     //声明c是一个指向Int类型指针的指针.
指针的指针 = & 指针
**c = &b

指针参数

当我们把一个指针做为参数传一个方法时，其实是把指针的复本传递给了方法，也可以说传递指针是指针的值传递。

如果我们在方法内部修改指针会出现问题，在方法里做修改只是修改的指针的copy而不是指针本身，原来的指针还保留着原来的值。

下面例子说明这个问题

void func1(int *p)
{
    int c = 2;
    p = &c;
}

int main(int argc, const char * argv[]) 
{
    int a = 1;
    int *b = &a;
    cout << "b pre  : " << *b << endl;
    func1(b);
    cout << "b next : " << *b << endl;
}

输出结果

b pre  : 1 
b next : 1

原因是传入的指针只是形参，改变后对原指针无影响。

指针的指针参数

void func2(int **p)
{
    int c = 2;
    *p = &c;
}

int main(int argc, const char * argv[]) 
{
    int a = 1;
    int *b = &a;
    cout << "b pre  : " << *b << endl;
    func1(&b);
    cout << "b next : " << *b << endl;
}

输出结果

b pre  : 1 
b next : 2

我们看一下 func2(int **p)这个方法

  • p:是一个指针的指针，在这里我们不会去对它做修改，否则会丢失这个指针指向的指针地址。
  • p: 是被指向的指针，是一个地址。如果我们修改它，修改的是被指向的指针的内容。换句话说，我们修改的是main()方法里 b指针

通过以上的小例子也就明白为什么C++中给链表进行操作时传递的是头结点的指针的指针了。

指针的指针的应用

下面是个向单链表中添加元素的方法，传入的参数是指针的指针。

void AddToTail(ListNode **pHead, int value)
{
    ListNode *pNew = new ListNode();
    pNew->m_nValue = value;
    pNew->m_pNext = NULL;

    if (*pHead == NULL)
    {
        *pHead = pNew;
    }
    else
    {
        ListNode *pNode = *pHead;

        while (pNode->m_pNext != NULL) {
            pNode = pNode->m_pNext;
        }

        pNode->m_pNext = pNew;
    }
}

总结

指针的用处：用于指向与其类型相同的地址，重点在于指向地址，而不在于指针本身的内容

一句话：指针a是一个地址, *a是它指向的内容，&a是它本身的内容
